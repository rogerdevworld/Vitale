04 — Tecnología

Stack técnico
seleccionado.

Mobile
React Native + Expo
iOS & Android desde un solo código. Acceso a cámara, notificaciones push y sensores nativos.
Web Admin & Pública
React + Next.js
Dashboard de profesionales en Vite+React. Web pública con Next.js para SEO y rendimiento.
Backend & Base de Datos
Supabase
PostgreSQL + Auth + Storage + Realtime. Sin servidor propio. Row Level Security para datos médicos.
Inteligencia Artificial
Claude API
Análisis de fotos de comida, planes nutricionales, asistente conversacional y corrección de técnica.
Pagos
Stripe
Suscripciones recurrentes, pagos únicos, facturación automática y gestión de reembolsos.
Monorepo & Build
Turborepo
Un solo repo para mobile, web y packages compartidos. Build incremental y caché inteligente.
Animaciones
Lottie + Skia
Animaciones 2D/3D de ejercicios en la app. Fluidas, ligeras y controlables por código.
CI/CD & Control de versiones
GitHub Actions + EAS
Deploy automático, builds de Expo para TestFlight y Play Store desde GitHub.
05 — Equipo médico

Validación humana
detrás de la IA.

La IA hace el primer análisis. El profesional humano valida, ajusta y firma cada plan. Ninguna recomendación de salud es 100% automática.

🥗
Nutricionista

Valida dietas, revisa macros, aprueba planes personalizados y atiende casos complejos.

💪
Entrenador Personal

Diseña rutinas, corrige técnica y adapta cargas según el progreso real del atleta.

🦴
Fisioterapeuta

Gestiona lesiones, diseña recuperaciones y libera o restringe ejercicios según estado físico.

🧠
Psicólogo

Trastornos de alimentación, motivación, ansiedad deportiva y bienestar emocional.

🩺
Médico Supervisor

Validación clínica general, revisión de perfiles y alta médica para programas intensivos.
